# Support

## Where To Ask Questions

- GitHub Issues for bugs and actionable requests
- GitHub Discussions for usage questions and implementation ideas

## Response SLO

- Issue first response target: within 48 hours (median)
- PR initial review target: within 72 hours (median)

These are operational targets, not guarantees.

## What To Include In Support Requests

- Library version
- Node version
- Example model URL (if shareable)
- `reasonCode` and `diagnostics` output
- Minimal reproduction code

## Project Boundaries

- This is an unofficial interoperability project.
- Maintainers cannot provide legal guidance on third-party ToS.
