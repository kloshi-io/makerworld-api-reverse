# v0.1.0 Release Notes

## Highlights

- Initial public release of `makerworld-api-reverse`.
- API-first MakerWorld resolver with typed fallback to `__NEXT_DATA__`.
- Model downloader with format and size validation.
- Non-throwing typed outcomes for integration-safe error handling.
- Community scaffolding: contributing, security, support, issue/PR templates.

## Included Examples

- `examples/basic-resolve.ts`
- `examples/download-model.ts`
- `examples/batch-import.ts`

## Known Limitations

- Upstream payload changes can require parser updates.
- Compatibility mapping is heuristic and may require model-specific adjustments.
- This is unofficial and not affiliated with MakerWorld/Bambu Lab.

## Upgrade Notes

This is the baseline release. No migration required.
