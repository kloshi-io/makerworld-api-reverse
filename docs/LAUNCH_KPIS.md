# Launch KPIs (First 30 Days)

Track weekly and cumulative:

- GitHub stars
- GitHub forks
- Opened issues
- Merged pull requests
- Unique contributors
- npm downloads (weekly)
- Median first response time on issues
- Median PR review turnaround time

## Suggested Targets

- 100+ stars
- 10+ contributors
- <= 48h first issue response median
- >= 90% CI pass rate on merged PRs
